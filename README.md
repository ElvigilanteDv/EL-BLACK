## 💻 𝗚𝘂𝗶𝗮 𝗱𝗲 𝗜𝗻𝘀𝘁𝗮𝗹𝗮𝗰𝗶𝗼𝗻 𝗲𝗻 𝗧𝗲𝗿𝗺𝘂𝘅

Sigue estos pasos para instalar y ejecutar **ELYSSIA BOT MD** en tu dispositivo.

[![Termux](https://img.shields.io/badge/Instalar%20Termux-000000?style=for-the-badge&logo=android&logoColor=white)](https://f-droid.org/es/packages/com.termux/)

> [!IMPORTANT]
> **No garantizamos un funcionamiento perfecto en Termux. Es normal que el bot funcione con lentitud o errores,🌸.**

1. **Concede Permisos de Almacenamiento**
```bash
termux-setup-storage
```

2.  **Actualiza y Prepara el Entorno**
    ```bash
    apt update && apt upgrade -y
    pkg install -y git nodejs ffmpeg imagemagick
    ```

3.  **Clona el Repositorio del Bot**
    ```bash
    git clone https://github.com/AmilcarGit/ElyssiaBot-MD.git
    ```

4.  **Accede al Directorio del Proyecto**
    ```bash
    cd ElyssiaBot-MD
    ```

5.  **Instala las Dependencias de Node.js**
    ```bash
    npm install
    ```
6.  **Mejora el Rendimiento del Bot**
    ```bash
    "start": "node --expose-gc index.js"
    ```

7.  **Inicia el Bot**
    ```bash
    npm start
    ```
🛠️ Requisitos
Node.js 18 o superior
Git
Conexión a Internet estable
